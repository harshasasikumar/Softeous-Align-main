Softeous-Align

A modern HR Solutions website built using Next.js and Tailwind CSS.
This project showcases key HR functions such as recruitment, onboarding, employee engagement, and people analytics in a clean, responsive UI.

Features

Multi-page layout: Home, HR, Services, About, Careers, Testimonials, Contact

Smooth navigation with Next.js App Router

Interactive UI with Tailwind CSS and Framer Motion animations

Fully responsive design for desktop, tablet, and mobile

Demo contact form and job listings

Tech Stack

Next.js 16 (App Router)

Tailwind CSS

Framer Motion for animations

React components

Getting Started

Clone the repo

git clone https://github.com/harshasasikumar/Softeous-Align-main.git


Install dependencies

cd Softeous-Align-main
npm install


Run the development server

npm run dev


Open http://localhost:3000
 in your browser
